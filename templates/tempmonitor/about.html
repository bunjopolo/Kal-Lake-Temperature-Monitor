About Kal Lake Temperature

Welcome to the Kal Lake Temperature website! This site provides you with temperature data for Kalamalka Lake in Coldstream BC, Canada. The charts on the homepage display the temperature trends and monthly average temperatures for the lake starting in August 2023.

The temperature data is collected from a remote sensor built with a Particle Boron cellular development board that records temperatures twice daily. You can find the source code for this website on my GitHub Repository.
