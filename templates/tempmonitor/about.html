About Kal Lake Temperature

Welcome to the Kal Lake Temperature website! There is no public temperature data that I'm aware of for Kalamalka Lake in Coldstream BC, Canada so I made this site to provide that information for cold water swimmers and anyone else that is interested. The charts on the homepage display the temperature trends and monthly average temperatures for the lake starting in August 2023.

The temperature data is collected from a remote sensor built with a Particle Boron cellular development board that records temperatures twice daily at the north end of the lake. You can find the source code for this website on my GitHub Repository.

For any inquiries or feedback, please contact me at kallakeweatherstation@gmail.com.

If you find this website useful and would like to support the app to help cover hosting and database fees to keep the server running you can contribute via the donation button in the menu bar!
